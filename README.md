# Estudos da Faculdade

Este repositório foi criado para organizar meus estudos, exercícios e aprendizados ao longo da graduação em Ciência da Computação.

## Disciplinas
- Algoritmo e Laboratório de Programação II
- Arquitetura e Organização de Computadores
- Geometria Analítica

## Objetivo
Registrar minha evolução acadêmica e prática, reunindo atividades, anotações, exercícios, códigos e trabalhos desenvolvidos durante o semestre.

## Organização
Os conteúdos deste repositório serão separados por disciplina, para facilitar a visualização e o acompanhamento dos estudos.

## Observação
Novas disciplinas e materiais poderão ser adicionados conforme a necessidade ao longo do curso.
